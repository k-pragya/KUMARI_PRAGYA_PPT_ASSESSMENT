Task 8: Create IAM Users and Groups

Objective::

Manage AWS access by creating IAM users and groups with proper permissions.

Steps::

Create Group: IAM → Groups → Create → Name it (e.g., AdminGroup) → Attach AmazonFullAccess → Save.
Create Users: IAM → Users → Add User → Console access → Set custom password → Require reset.
Add to Group: Add users to AdminGroup during or after creation.
Verify: Ensure users appear in IAM and inherit group permissions.
