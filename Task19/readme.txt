This document describes the process of creating and exploring storage services within the Microsoft Azure Portal. A Storage Account named pptstorage123 is provisioned under the existing Resource Group with default configuration settings and selected region. After successful deployment, the Storage Account dashboard is accessed to manage storage services.
A new container is created under Blob Storage, and a sample file is uploaded to demonstrate object storage functionality. Additionally, the File Shares section is explored to understand managed file storage capabilities, and the Tables section is reviewed to examine NoSQL key-value storage options. This exercise provides practical exposure to Azure Blob, File, and Table storage services.
