This document outlines the procedure to configure a basic networking setup in the Microsoft Azure Portal. A Virtual Network named ppt-vnet is created within the existing Resource Group ppt to enable secure communication between Azure resources. The address space is defined as 10.0.0.0/16 to provide a structured IP allocation range. Within the Virtual Network, a subnet named subnet1 is configured with the address range 10.0.1.0/24 to logically segment network resources. After deployment, the Virtual Network dashboard is accessed to verify the subnet configuration and ensure proper IP addressing. This setup establishes the foundational networking infrastructure required for hosting and connecting Azure services securely.
